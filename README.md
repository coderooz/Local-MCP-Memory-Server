# Local-MNC-Memory-Server

A persistent memory system for AI agents using the Model Context Protocol (MCP).

## Features

- Multi-agent memory
- Cross-project context sharing
- Action tracking
- Error logging
- MongoDB-backed persistence

## Tools

- store_context
- search_context
- log_action
- get_full_context
- start_session

## Setup

```bash
npm install
node server.js
node mcp-server.js
```
